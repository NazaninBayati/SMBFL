News

Version 0.6 of JTopas is out. A fairly deep redesign has been applied to the library. Management of the more static tokenizer properties like keywords, comments etc. are separated from the actual tokenizing process. Multible tokenizers can now share the same instance containing these properties. Beside the better design, the library integrates much better in multithreaded environments.

Examples for the older versions of JTopas can still be found here.
