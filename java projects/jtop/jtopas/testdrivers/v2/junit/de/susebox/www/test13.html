Project Description

Popular web servers must deal with huge number of client requests and are often the bottleneck in accessing information supplied by them. Push and pull technologies, caching, clustering, replication and consistency are some of the issues that need to be explored in building scalable servers. This project will explore these techniques for dynamic content and will study workloads that can be used to evaluate their effectiveness.

Deliverables

A project report that identifies the problems discussed, solutions provided, and my views on the problems, solutions.
